# vue_shop
### Import db
```Import db
import server/db/my.db
```
## Project setup
```
npm install
```

### Start node server
```
cd server
node ./app.js
```

### Compiles and minifies for production
```
npm run build
```

### Compiles and hot-reloads for development
```
npm run serve
```
